# Glass Eyewear E-commerce

Website bán kính mắt hiện đại với PWA, SSR, AI thử kính ảo.

## Tech Stack
- Next.js 14+ (App Router, SSR/CSR hybrid)
- MongoDB + Prisma ORM
- TypeScript
- TipTap Editor
- Gemini AI (thử kính ảo)
- ChatGPT (auto content)

## Getting Started

```bash
npm install
npx prisma generate
npx prisma db push
npm run dev
```

## Environment Variables
Copy `.env.example` to `.env` and fill in your values.
